Welcome to the FangEngine!

This engine allows you to easily create object oriented, event-driven games in Python! You can customize as much or as little of the engine as you need to build your next great game!

To get started, simply run the following command in your terminal: pip install FangEngine

Now that everything is installed, you can check out the FangEngine Documentation or the Issue Tracker

Thank you for using FangEngine, and good luck with your next project!

Documentation

This wiki contains instructions on performing specific tasks. For the full documentation, please visit the GitHub Pages.
